# نَبرة – الهوية بالبصمة الصوتية

**نَبرة** هو نموذج تجريبي (Proof of Concept) للتحقق من الهوية باستخدام **بالبصمة الصوتية**، يهدف إلى تسهيل  الدخول للخدمات الرقمية من خلال الصوت بدلًا من كلمات المرور أو الخطوات المعقدة.

تستند فكرة المشروع إلى تمكين المستخدم من إثبات هويته باستخدام صوته فقط، مما يسهم في تحسين تجربة المستخدم، خصوصًا كبار السن أو المستخدمين الذين يواجهون صعوبة في استخدام الطرق التقليدية.

---

## فكرة المشروع

يقوم المستخدم بتسجيل صوته مرة واحدة عند التسجيل الأولي.  
يتم تحليل خصائص الصوت واستخراج **بصمة صوتية فريدة** تمثل هوية المستخدم، ثم يتم حفظها بشكل آمن.

عند كل محاولة دخول لاحقة، يقوم المستخدم بتسجيل صوته مرة أخرى، ويقوم النظام بمقارنة الصوت الجديد مع البصمة الصوتية المحفوظة للتحقق من الهوية.
